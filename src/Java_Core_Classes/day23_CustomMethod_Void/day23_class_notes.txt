declaring a method:

Access Modifier - Specifier - Return Type - Method Name (parameter) {
}

Access Modifier : public = always accessible, protected, default, private
Specifier: static = method can be called through its class name, final, abstract, synchronized
Return Type: void = does not return a value, any data type
Method Name: can be any name, meaningful, descriptive
Parameter: any data type

Custom Methods:
    1. void methods
    2. void methods with arguments
    3. return methods
    4. returns with argument
    5. overloading
